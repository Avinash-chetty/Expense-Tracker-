# Expense Tracker (Python)

## üìå Project Description

This is a **simple console-based Expense Tracker** built using **Python**. It allows users to record daily expenses, view all expenses, and calculate total spending. This project is beginner-friendly and is designed as a step into Python programming, data handling, and logic building.

The project uses **lists and dictionaries** to store expense data and provides a menu-driven interface for easy interaction.

## üß† Python Concepts Explained

### 1Ô∏è‚É£ Variables and Data Types

* Variables are used to store data such as user choices, dates, categories, and amounts.
* Different data types are used:

  * `int` for menu choice
  * `float` for expense amount
  * `string` for date, category, and description

### 2Ô∏è‚É£ Lists

* A list named `expensesList` is used to store all expenses.
* Each expense is added to the list, allowing multiple records to be saved during runtime.

### 3Ô∏è‚É£ Dictionaries

* Each expense is stored as a dictionary.
* The dictionary contains key-value pairs such as:

  * `date`

  * `amount`
* This helps organize expense data in a structured format.

### 4Ô∏è‚É£ Loops

* A `while True` loop is used to keep the program running until the user chooses to exit.
* A `for` loop is used to:

  * Display all expenses
  * Calculate the total spending

* `if-elif-else` conditions are used to handle menu choices.
* Each menu option triggers a different block of code.

### 6Ô∏è‚É£ User Input and Output
